Day 3 API Specifications

1. CoinGecko Public Markets API

Retrieves live market prices, capitalization ranks, 24-hour highs/lows, icons, and 7-day sparkline coordinates for top-tier digital assets.

GET https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd&ids=bitcoin,ethereum,solana,ripple,cardano,dogecoin,polkadot,chainlink&sparkline=true

2. Frankfurter exchange rates API

A keyless, open currency rate registry providing foreign exchange conversion parameters with USD as the anchor currency.

GET https://api.frankfurter.dev/v2/latest?base=USD
