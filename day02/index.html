Day 2 API Specifications

1. NASA Astronomy Picture of the Day (APOD)

Fetches the daily imagery/video with cosmic descriptions provided by astronomers.

GET https://api.nasa.gov/planetary/apod?api_key=DEMO_KEY

Fallback key: Handles empty values or media placeholders if limit is hit.

2. Where the ISS At? API

Returns the real-time geographical coordinates, velocity, and altitude of the ISS.

GET https://api.wheretheiss.at/v1/satellites/25544
