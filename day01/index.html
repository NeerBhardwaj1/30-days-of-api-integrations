Day 1 API Specifications

1. Client IP Geolocation API

Fetches geographical context, ISP coordinates, and timezone using the client's public IP address.

GET http://ip-api.com/json/

Fallback endpoint: https://ipapi.co/json/ (for SSL support).

2. Open-Meteo Weather API

Free, keyless meteorological forecast API. Queries specific coordinate parameters extracted from the Geolocation API.

GET https://api.open-meteo.com/v1/forecast?latitude={lat}&longitude={lon}¤t_weather=true&daily=temperature_2m_max,temperature_2m_min,weathercode&timezone=auto
