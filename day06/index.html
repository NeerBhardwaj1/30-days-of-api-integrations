Day 6 API Specifications

1. Jikan API (MyAnimeList)

A completely free, keyless, and open-source REST API providing access to the extensive MyAnimeList anime and manga directory, support for currently airing seasons, and general searches.

GET https://api.jikan.moe/v4/seasons/now?limit=15
GET https://api.jikan.moe/v4/anime?q={query}&limit=15

2. Studio Ghibli API

A community-maintained REST mirror (ghibliapi.dev) returning classic film metadata, Rotten Tomatoes scores, running durations, TMDb poster links, and banner images.

GET https://ghibliapi.dev/films
