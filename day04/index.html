Day 4 API Specifications

1. The Cat API

A public keyless registry providing randomized cat imagery, dimensions, and optional breed specifications.

GET https://api.thecatapi.com/v1/images/search

2. Dog CEO API

A comprehensive open database of dog breeds and sub-breeds returning random photos categorized by structural path strings.

GET https://dog.ceo/api/breeds/image/random
